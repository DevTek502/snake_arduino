# Arduino  Mega
Pines de entrada
* pin 18: Boton Derecho
* pin 19: Boton Izquierda
* pin 20: Boton Arriba
* pin 21: Boton Abajo

Pines de Salida
* pin 12: DIN
* pin 13: CLK
* pin 14: CS

---
# MAX7219
Pines de entrada:  
* pin 1: DIN
* pin 12: CS
* pin 13: CLK
* pin 18: VCC  

Opcional:
* pin 24 (DOUT): conexion en cascada para otro MAX7219

# Configuracion en Proteus


 Se realizo una configuracion para un mejor uso del joystick en proteus utilizando botones del teclado.

>## Atajos de teclado:
  * Arriba ↑    ~   Numeral 2
  * Abajo ↓     ~   Numeral 3
  * Derecha →   ~   Numeral 4
  * Izquierda ← ~   Numeral 1
